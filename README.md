# dnsmasq

容器化运行，仅有8.97MB，简单方便，自带去广告功能

![编译状态](https://github.com/lostmaniac/dnsmasq/workflows/Github%20Build%20Image/badge.svg?branch=master)

## 支持多CPU架构

* linux/386
* linux/amd64
* linux/arm/v6
* linux/arm/v7
* linux/arm64
## 使用方法

本机安装git和docker

```git clone https://github.com/lostmaniac/dnsmasq.git```

```cd dnsmasq```

```docker-compose up -d```
